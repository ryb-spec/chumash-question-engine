# Perek 3 Short Re-Pilot Manual Scope Checklist - 2026-04-29

## Purpose

Use this checklist while running the short Perek 3 re-pilot. This is necessary because this task did not add a runtime-enforced short-repilot filter.

## Before starting

- Confirm Learn Mode is being used.
- Confirm the student count is 1-2.
- Confirm the target length is 8-10 questions per student.
- Keep the teacher monitor or pilot monitor open.

## In-scope evidence

Count observations only for:

- revised tense/form wording
- revised prefix prompt wording
- `דֶּרֶךְ` translation distractor repair
- `אֲרוּרָה` translation distractor repair

## Excluded from evidence

Do not count these as clean short re-pilot evidence:

- `אָשִׁית` / `שית` shoresh question
- unverified `phrase_translation` questions
- Perek 4 content
- any runtime expansion or promotion claim

## Stop or skip conditions

If any of the following appear, record a scope leak:

- Prompt: `What is the shoresh of אָשִׁית?`
- Expected answer or answer choice: `שית` as a beginner shoresh target
- `question_type=phrase_translation`
- Bereishis Perek 4 content
