# Loshon Hakodesh Source Map

## What the Main Textbook/Workbook Is

The main instructional source is the Loshon Hakodesh / Lashon Hakodesh workbook-textbook pair file currently observed as:
- `Loshon Hakodesh Book OCR.pdf`

For a future approved private/internal storage workflow, the preferred repo-local protected filename would be:
- `docs/sources/loshon_hakodesh/raw/loshon_hakodesh_mastering_the_basics_textbook_ocr.pdf`

This source provides:
- lesson sequence
- rule explanations
- dikduk teaching order
- exercise prompts
- vocabulary and grammar framing

## What the Answer Booklet Is

The full companion answer source is:
- `loshon answe 2-combined.pdf`

For a future approved private/internal storage workflow, the preferred repo-local protected filename would be:
- `docs/sources/loshon_hakodesh/raw/loshon_hakodesh_mastering_the_basics_answer_booklet_full.pdf`

A later reinspection found a cleaner OCR-readable 33-page copy of this answer booklet, and that cleaner copy is now the preferred teacher-only answer-key metadata source.

This source provides:
- expected exercise answers
- teacher-key support
- validation targets for generated-answer QA
- lesson-aligned review guidance

## Why Both Must Be Kept Together

These files form a matched instructional source pair:
- the textbook/workbook contains the rule sequence and exercises
- the answer booklet contains the expected answers and exercise validation layer

Using only one side would be incomplete:
- textbook-only loses authoritative answer alignment
- answer-only loses instructional rule context

## How the Answer Booklet Should Be Used

Use the answer booklet for:
- teacher-only review
- generated-answer validation
- reviewer notes
- answer expectation checks
- lesson-to-exercise alignment work

## How the Answer Booklet Must Not Be Used

Do not use the answer booklet as:
- student-facing runtime content
- a public answer bank
- a direct student handout
- a production-generated answer source without independent review

## Relationship to Zekelman Standards

This source pair is especially relevant to:
- Zekelman Standard 3: Vocabulary and Language Skills

It can help with:
- parts of speech
- prefixes and suffixes
- verb forms
- סמיכות
- prepositions
- direct-object handling
- agreement and syntax
- numbers and form changes

The Loshon Hakodesh pair is supplemental instructional structure, not a replacement for the canonical Zekelman standards wording.

## Relationship to the Dikduk Rules Engine

This source pair is a strong candidate foundation for:
- lesson-by-lesson rule extraction
- rule IDs
- exercise-to-rule mapping
- confusion pattern modeling
- teacher-only validation examples

## Relationship to Generated-Question QA

The textbook/workbook can help identify:
- which rule is being taught
- what exercise shape is being modeled

The answer booklet can help validate:
- whether a generated answer is plausible
- whether an exercise interpretation stays aligned to the source

Any generated question or answer output still requires independent review before runtime use.
